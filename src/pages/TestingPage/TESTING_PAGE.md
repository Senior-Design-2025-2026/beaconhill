# Testing Page
This page is used to test feature implementation. The creation of components shall be tested by placing them on the testing page. The developer will then observe its appearance and functionality. 

## Page Structure
This page will contain a list of simple component tests, similar to an interactive storybook. Each component will contain a section that has simple input to modify the visual component. 

Examples of tests:
- button that creates dummy data
- slider
- linear gauge
